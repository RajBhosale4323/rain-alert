# Auto Rain Alert System

A Python automation project that checks weather forecasts using the OpenWeather API and sends rain alerts through:

- Email
- SMS
- WhatsApp

## Features
- Real-time weather forecast checking
- Rain detection using weather condition codes
- Email notifications using SMTP
- SMS alerts using Twilio
- WhatsApp messaging integration
- Automated alert system

## Technologies Used
- Python
- Requests
- SMTP
- Twilio API
- OpenWeather API
